  ## How to use
  1. 點擊右上方鼠鬚管圖標 > Settings
  2. 將除了font以外所有文件貼入目前文件夾
  3. 點擊右上方鼠鬚管圖標 > Deploy
  4. 切換成其他輸入法
  5. 安装字体文件(font 文件夹内的两个文件)
